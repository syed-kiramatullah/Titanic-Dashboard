# 🚢 Titanic Excel Dashboard

## 📌 Project Overview

This project presents an end-to-end data analysis of the Titanic dataset using Microsoft Excel.
The goal is to analyse passenger data and identify key factors that influenced survival.

---

## 🎯 Objectives

* Analyse survival patterns of passengers
* Identify the impact of gender, class, fare, and age
* Build an interactive dashboard using Excel

---

## 🧹 Data Cleaning

* Handled missing values (Age, Embarked)
* Removed unnecessary columns
* Created new features:

  * Age Group
  * Family Size
  * Is Alone

---

## 📊 Key Performance Indicators (KPIs)

* Total Passengers
* Total Survivors
* Survival Rate (%)
* Average Fare (Survived vs Not Survived)

---

## 📈 Dashboard Features

* Survival analysis by Gender
* Survival analysis by Passenger Class
* Fare-based survival insights
* Age group analysis
* Interactive slicers (Gender, Class, Embarked)

---

## 🔍 Key Insights

* Female passengers had a significantly higher survival rate than males
* 1st class passengers were more likely to survive
* Higher fare passengers had better survival chances
* Children had relatively higher survival rates

---
